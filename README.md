A real-time Endorsement Web App built with HTML, CSS, JavaScript, and Firebase Realtime Database.
Users can write endorsements, publish them instantly, and delete them in real time.

📌 Overview
This project allows users to:
Write an endorsement message
Send it To someone
Sign it From themselves
Publish it to a live database
View endorsements instantly
Delete endorsements by clicking on them
All data is stored and synced using Firebase Realtime Database.

🚀 Features
Real-time database updates
Add new endorsements
Delete endorsements
Dynamic UI rendering
Auto-updating list from Firebase
Clean and responsive design

🛠 Technologies Used
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6 Modules) – Logic
